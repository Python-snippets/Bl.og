## Running Bl.og (locally)
### clone this repo 
```
https://github.com/Python-snippets/Bl.og.git
```

### Install Django 
```
pip3 install django
```

### Prepare your environment
```
python3 -m venv env
```

``` 
source env/bin/activate
```

### Run The development server
```
python manage.py runserver
```
